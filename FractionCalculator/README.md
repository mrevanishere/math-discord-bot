# Fraction Calculator

## Specifications
### Input
* Input will be in the form of a value with either a fraction or an integer

* Term followed by an operator

* Multiple operators / expressions supported

* Parenthesis / Brackets / Braces unsupported
### Output
* Out is reduced and in improper fraction form

### Examples
| Input             | Output    | Notes                                         |
|-------------------|-----------|-----------------------------------------------|
| `1/4 + 1_1/2`     | `1_3/4`   |                                               |
| `8/4 + 2`         | `4`       | Input may be an improper fraction.            |
| `-1 * -1/2`       | `1/2`     |                                               |
| `-11/17 + -1/17`  | `-12/17`  |                                               |
| `0 * 25_462/543`  | `0`       | Remember to check for border/special cases.   |

#### Multiple Operations

| Input                     | Output    |
|---------------------------|-----------|
| `1 + 3 + 3`               | `6`       |
| `3/4 * 4`                 | `3`       |
| `-1/2 * 4 + 3/4`          | `-1_1/4`  |
| `5_3/4 - -6_8/8 - 5_3/4`  | `7`
